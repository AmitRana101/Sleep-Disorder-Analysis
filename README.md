💤 Sleep Disorder Analysis Project
📌 Overview

This project focuses on analyzing sleep patterns and identifying factors related to sleep disorders using SQL. The dataset contains information about individuals’ lifestyle habits, health metrics, and sleep behavior.

The goal of this project is to extract meaningful insights such as how stress, occupation, physical activity, and health conditions impact sleep quality and disorders like insomnia and sleep apnea.

🗂️ Database Setup

Imported dataset: sleep_disorderdataset
Cleaned and standardized column names for better readability and query efficiency
